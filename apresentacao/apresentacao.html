JURASHOW

Système de réservation d'attractions dinosaures développé avec Node.js, TypeScript et PostgreSQL

Introduction au Projet

JuraShow est une application web de réservation d'attractions sur le thème des dinosaures. Le projet permet aux visiteurs de découvrir différentes attractions dinosaures et de réserver leurs billets en ligne.

Objectif du Projet

Créer un système complet de gestion des réservations avec une interface utilisateur intuitive et une base de données robuste pour gérer les attractions, les billets et les réservations.

Thématique

Chaque attraction représente un dinosaure différent :

  • Vélociraptor - L'attraction de la vitesse et de l'intelligence
  • Tyrannosaure - Le roi des dinosaures
  • Ptérosaure - L'expérience aérienne
  • Thérizinosaure - Le géant aux griffes impressionnantes
  • Tricératops - La rencontre paisible
  • Plésiosaure - L'aventure aquatique

Architecture MVC

Le projet suit le pattern Model-View-Controller pour une séparation claire des responsabilités :

MODEL

Rôle : Gestion des données

Fichiers :

  • Database.ts
  • ticketModel.ts
  • reservationModel.ts
↔

VIEW

Rôle : Interface utilisateur

Fichiers :

  • Templates EJS
  • CSS/JavaScript
  • Assets statiques
↔

CONTROLLER

Rôle : Logique métier

Fichiers :

  • homeController.ts
  • ticketController.ts
  • reservationController.ts

Flux de Données MVC

1. Requête utilisateur

L'utilisateur effectue une action via l'interface (View)

2. Routage

Express.js dirige la requête vers le bon Controller via les Routes

3. Traitement

Le Controller traite la logique métier et fait appel aux Models

4. Accès aux données

Les Models interagissent avec la base de données PostgreSQL

5. Rendu de la réponse

Le Controller renvoie les données à la View qui génère le HTML final

Opérations CRUD

L'application implémente les opérations CRUD complètes pour la gestion des réservations :

CREATE - Création de Réservations

Route : POST /reservations

Controller : ReservationController.createReservation()

Model : ReservationModel.createReservation()

static async createReservation(data: CreateReservationData): Promise<number> { const result = await query(` INSERT INTO reservation (user_id, reservation_id, name_buyer, buyer_email, date, quantity, ticket_id, total_price, payment) VALUES ($1, $2, $3, $4, $5, $6, $7, $8, $9) RETURNING reservation_id `, [userId, reservationId, ...data]); }

READ - Lecture des Données

Routes : GET /reservations/api/:id, GET /tickets

Méthodes :

  • ReservationModel.getReservationById()
  • ReservationModel.getReservationsByEmail()
  • TicketModel.getAllTickets()
  • TicketModel.getTicketById()

UPDATE - Mise à Jour

Route : PUT /reservations/api/:id

Controller : ReservationController.updateReservation()
